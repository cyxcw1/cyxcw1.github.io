*   [Askime](http://akismet.com/ "Askime")：能够防止垃圾评论的强大插件。
*   [Count Per Day](http://www.tomsdimension.de/wp-plugins/count-per-day)：统计插件，能够统计访客数量，阅读数量，能够显示IP等信息，功能比较强大。
*   [Crayon Syntax Highlighter](https://github.com/aramk/crayon-syntax-highlighter)：代码高亮插件，最近比较流行的一款，个性化功能很多。
*   [Google XML Sitemaps](http://www.arnebrachhold.de/projects/wordpress-plugins/google-xml-sitemaps-generator/)：Google的站点图插件，这边方便其他用户使用Chrome浏览器搜到你的网站，每次更新文章会该插件会自动更新XML等信息。
*   [JiaThis分享工具](http://www.jiathis.com/help/html/wordpress-install-jiathis)：社交分享工具，方便读者分享你的网站以及文章。
*   [Rounded Tag Cloud](http://suhanto.net/rounded-tag-cloud-widget-wordpress/)：彩色标签云插件。
*   [WP-PostViews](http://lesterchan.net/portfolio/programming/php/)：统计每篇文章阅读量的插件。
